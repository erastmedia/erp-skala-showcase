# Security Policy

This repository is a public showcase repository and does not contain production credentials, private infrastructure configuration, or complete production source code.

## Reporting Security Concerns

If you notice an accidentally exposed secret, sensitive path, private business data, or security-related issue in this repository, please contact:

- Email: erastmedia@gmail.com

## Public Repository Rules

The following content must never be committed here:

- `.env` files
- API keys
- SMTP credentials
- database credentials
- server IP/private deployment details
- production logs
- SQL dumps
- customer data
- real payroll data
- real business financial data
- private roadmap or pricing strategy
