# Public Roadmap

This roadmap is intentionally high-level. It avoids exposing private product strategy.

## Current Focus

- Stabilize ERP core workflows.
- Improve mobile usability.
- Strengthen documentation.
- Prepare demo and portfolio materials.
- Validate MSME-friendly workflows.

## Near-term Direction

- UX simplification for non-technical users.
- Better onboarding and empty states.
- More polished dashboard summaries.
- More complete finance and inventory workflows.
- More demo-ready screenshots and walkthrough videos.

## Future Direction

- Web admin portal using the same API.
- OCR/AI-assisted document input.
- More advanced reporting.
- Subscription and tenant management.
- Integrations with external services.

## Product Principle

Automation should accelerate business records, not replace user review. The system must remain understandable and usable even without AI.
